# Hopsule Scoop Bucket

Official [Scoop](https://scoop.sh/) bucket for Hopsule CLI tools.

## Installation

```powershell
scoop bucket add hopsule https://github.com/Hopsule/scoop-bucket
scoop install hopsule
```

## Update

```powershell
scoop update hopsule
```

## Available Tools

| Manifest | Description |
|----------|-------------|
| `hopsule` | Decision-first, context-aware, portable memory system CLI |
